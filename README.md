# Kitap Katalog Uygulaması

Bu proje, Flutter eğitimi kapsamında geliştirilmiş basit bir kitap katalog ve sepet uygulamasıdır.

## Proje Özellikleri
- Kitap listeleme
- Kitap detay sayfası
- Sepete ekleme
- Toplam fiyat görüntüleme

## Kullanılan Teknolojiler
- Flutter
- Dart
- Material Design

## Flutter Sürümü
Flutter 3.41.7

## Çalıştırma Adımları
1. Projeyi bilgisayarınıza indirin
2. Terminalde proje klasörüne girin
3. `flutter pub get` komutunu çalıştırın
4. `flutter run` komutunu çalıştırın

## Ekran Görüntüleri
Proje içinde screenshots klasörüne eklenmiştir.
